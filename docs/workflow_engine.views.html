workflow_engine.views.enqueueable_job_grid module¶

class workflow_engine.views.enqueueable_job_grid.EnqueueableJobGrid(row_range, order=None)[source]¶

Bases: object

Queries for a subset of enqueued objects (rows), workflow nodes (columns) and Jobs (grid entries) with information to display a grid view of progress through the workflow. While this class may be used alone for simple workflows, a complex job grid with multiple enqueued object types should subclass it.

See also the workflow_engine.views.faster_job_grid.FasterJobGrid view for how to call it and the job_grid.js javascript client for rendering.

Notes

Dataframes representing partial calculations are saved as data members so they can be printed for debugging. This may change in the future.
